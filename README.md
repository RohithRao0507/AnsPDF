# PDF Answer Bot 
Add your pdf document and start asking questions, this would help in finding the relevant answers. This works using the OpenAI GPT-3.0's intelligence and Langchin Framework along with similarity search. 

### Software Used
  - GPT-3.0
  - LangChain
  - Python
  - PyPdf reader

In the above folder, I have tried it on different types of Laundry appliances (washers, dryers) to recognize the repair action, fault codes and issues given by the user input.
